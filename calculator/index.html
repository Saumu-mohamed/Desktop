# Interactive Web Calculator ## Overview This is a fully functional, responsive web-based calculator built with HTML, CSS, and JavaScript. It supports basic arithmetic operations, decimal calculations, parentheses, memory functions, and a calculation history panel. The calculator features a light/dark mode toggle and keyboard support for enhanced usability. ## Features - **Basic Operations**: Addition (+), subtraction (–), multiplication (×), division (÷). - **Advanced Features**: - Parentheses for grouping expressions (e.g., `(5 + 3) × 2`). - Memory functions: M+ (add to memory), M- (subtract from memory), MR (recall memory), MC (clear memory). - Calculation history panel to log completed calculations. - Light/dark mode toggle using CSS variables. - Keyboard support for numbers, operators, Enter (calculate), Backspace, and Escape (clear). - **Responsive Design**: Adapts to desktop and mobile viewports. - **Error Handling**: Prevents invalid inputs (e.g., multiple decimals, consecutive operators) and displays errors for invalid calculations. - **Styling**: Grid-based button layout, hover effects, active-press animations, and dynamic font sizing for large
